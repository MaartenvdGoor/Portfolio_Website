    Acclivity

    2D platformer

    Acclivity is a 2D platform game made for android. The goal of the game is to solve different types of puzzles. Made by 2 people as part of a college assignment. I was responsible for the movement system and level design. The project was done in C# with the Unity game engine. I learned much about what makes a game challenging but fun to play. I also learned a lot about development itself. How to make a character move satisfyingly with mobile controls
