    About this project

    Balls

    Game development

    Balls is a recreation of 2 of the most well-known retro games: Ball and Arkanoid. This was a solo project in unity using the C# language. With this project, I learned to use the same objects for different purposes instead of making multiple similar but different objects. I gained a greater understanding of the unity engine.
